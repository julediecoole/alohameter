# alohameter
Alohameter ist eine Webanwendung, die Wind-, Wellen- und Wetterdaten von vier hawaiianischen Inseln visualisiert. Sie kombiniert aktuelle Wetterdaten aus der Open-Meteo-API mit historischen Werten aus einer eigenen MySQL-Datenbank und bietet anschauliche Diagramme für Wellenhöhe, Wind und Temperatur im 48-Stunden-Verlauf.

Learnings:
Wir haben den Umgang mit Chart.js, dynamischer API-Einbindung und der Umsetzung responsiver, animierter Layouts gelernt. Besonders lehrreich war das Zusammenspiel von Frontend (HTML, CSS, JavaScript) und Backend (PHP, MySQL).

Schwierigkeiten:
Herausfordernd war insbesondere die korrekte Einbindung der Diagramme in Popups, das Responsive Design, das Gruppieren und Filtern der API-Daten sowie das Verhindern doppelter Eventlistener. Die Integration mehrerer APIs und Anpassungen bei Animationen, Diagrammgrößen und SVG-Elementen verlangten viele Iterationen.

Benutzte Ressourcen:
- Chart.js für interaktive Diagramme
- Open-Meteo-API für aktuelle Wetterdaten
- Eigene MySQL-Datenbank für historische Werte
- PHP als Schnittstelle zwischen Datenbank und Frontend
- HTML, CSS und JavaScript für das Interface
